overview

This is my website, but accessible over SSH. When you connect to port 22, you receive a version of my homepage at vedsite.com, but in the terminal. I tried to keep consistent to the DVD-player-esque animation style, with the shapes bouncing and hitting the edges. Each shape maps to a section of my portfolio (about, projects, experience, misc) and morphs into a different character every time it hits a wall. You can press 1 through 4 on your keyboard to navigate into each section, or just sit by and watch the shapes bounce around (don't sit idle too long, i have a timeout set for 10 minutes).

visuals

bouncing square navigation: four squares made up of four different icons (█ ◆ ● ▓) drift across the terminal, with each square color-coded to a specific page. pink = about, purple = projects, blue = experience, teal = misc. the numbers and colors match the ones on my website homepage respectively.
square morphing: when a square collides with the edge of the terminal window, its character icon cycles to the next in the set. purely cosmetic, but helps stay true to the original animation.
tiled background: staying true to the background on my homepage, i have my (clever) personal domain repeating across every cell in the terminal as a dim grid, providing a nice background to watch the squares bounce around in. the text is offset per row so it reads diagonally, and the squares render on top of it.
keyboard navigation: press 1–4 to open a subpage, b or esc to go back, q to quit. each subpage shows the section title and links back to the real website.
per-user sessions: every SSH connection spawns its own independent instance of the UI. your shapes, your session.

how it's built

the app is written in Go using the Charm stack — a set of libraries purpose-built for terminal UIs over SSH.

Go Wish Bubble Tea Lip Gloss Docker AWS EC2

Wish handles the SSH server — it listens on port 22 and bootstraps a fresh terminal UI for each incoming connection. Bubble Tea is the TUI framework that drives the render loop: an Init → Update → View cycle running at 50ms ticks for smooth animation. Lip Gloss handles all the styling — colors, bold text, positioning — in the terminal.

the animation loop works like a simple physics engine: each shape has an (x, y) position and a velocity vector (vx, vy). every tick, positions update and bounce off terminal edges. on collision, the shape's character morphs to the next in the cycle.

architecture

your terminal → SSH (port 22) → Docker → Wish server → Bubble Tea UI

the Go binary runs inside a Docker container on my AWS EC2 instance. port 22 on the host is mapped to the container, so ssh ssh.vedsite.com routes directly into the Wish server. my admin SSH runs on a different port so the portfolio can own port 22. Cloudflare handles DNS (A record, DNS-only mode — SSH can't be proxied).
